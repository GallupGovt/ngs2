
1. Go to https://raw.githubusercontent.com/GallupGovt/ngs2/master/cycle3/Polycraft/preregistration/analytics.R 
2. Save "analytics.R" file locally. 
3. Open "analytics.R" in R studio and execute. The script will download the remaining scripts from Github and run all the pre-registration analytics. 

NOTE: The simulation-based power analysis has a long runtime (~48 hours), and so it is commented out.
